## 技術的な工夫
Date.now()を用いて、遅延をナシに。
setIntervalが重複しないように、検出し、イベント冒頭でclearIntervalを実行。
フェードイン/アウト効果で、違和感がないテキスト切り替えに。

## 学んだこと
Setintervalは完璧ではない。
何故か。それはJavaScriptがシングルスレッドであることが起因で、1000ms後、入るべきタイミングである時に他のイベントが起きていたら、後回しになってしまう。故にそのズレは修正されることなく、どんどん数ミリ単位でズレていくのだ。
そのため、Date.now()を使った。
何故Date.now()ではズレが生じないのか、Date.now()はOSの時間を見る。故に、遅延が仮に起きてもそれを正しい時間に書き直すことが可能。つまり、Setintervalがズレても、Date.now()は実際の時間を返すため、その遅れを検出し、補正することが可能なのだ。

disabled属性の挙動

### キーボードショートカット

- **Space**：スタート/一時停止/リスタート
- **R**：リセット
- **L**：ラップ（タイマー動作中のみ）
